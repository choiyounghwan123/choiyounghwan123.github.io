# 백준 11720번 문제 풀이

안녕하세요! 이번 포스트에서는 백준 11720번 문제 "숫자의 합" 문제를 Python을 이용하여 해결해보겠습니다.



## 문제 설명

**문제 이름**: [숫자의 합 (백준 11720번)](https://www.acmicpc.net/problem/11720)



**문제 설명**: 

N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.



**입력**:

첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.



**출력**:

입력으로 주어진 숫자 N개의 합을 출력한다.

**예시**:

![example](/Users/choeyeonghwan1/Desktop/choiyounghwan123.github.io/assets/images/11720/example.png)



## 해결 방법

**아이디어**:

- 주어진 숫자열을 문자열로 받아 list로 변환시키면 각 인덱스마다 한자릿수를 가진다. 

     **예시**:

  ```python
  import sys
  
  nums = int(sys.stdin.readline().rstrip())
  
  print(N)
  ```

  **입출력**:

  ```python
  1234
  ['1', '2', '3', '4']
  ```

- nums배열의 각 원소들은 int로 변환하여 sum변수에 모두 더하여 출력하였다.

  


## 전체코드

```python
# 숫자의 합 구하기

import sys
N = int(sys.stdin.readline())
nums = list(sys.stdin.readline().strip())
sum = 0

for num in nums:
    sum += int(num)
print(sum)
```
